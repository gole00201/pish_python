## Выражения в Python
### Логические выражения

```python

    and # Логическое и
    not # Логическое не
    < # Меньше чем
    > # Больше чем
    == # Равно
    != # Не равно

    <= # Меньше либо равно
```

```python
    x = True
    y = False
    z = x or y  #---
    z = x and y # ---
    z = not(x and y) or (y or not x) #---
```

## Побитовые операции

```python
    << # Побитовый сдвиг влево
    >> # --`--- вправо
    | # Побитовое или
    ~ # Побитовое не
    & # Побитовое и
    ^ # Побитовое исключающее или
```

---

### Пример
```python
    x = 0
    x = x | (1 << 0) # ---
    x = 0
    x = x & ~(1 << 0) # --- (последние 16 бит)
    a = 4
    b = 2
    z = a ^ b & 0

```

## Функции в Python
Функции в python -  это объекты, которые принимают в качестве аргументов другие объекты и возвращают значение на выходе

```python
    def function(x):
        return x**x + x / 2
    
    def middle(a , b):
        return (a + b) / 2
    
    def get_user_name():
        name = input("Введите имя: ")
        return name

```
---

Функции в python можно присваивать, потомучто операция присваивания в Python это не копирования одного значения в другое, это передача **ссылки**.

```python
    def function(x):
        return x**x + x / 2
    x = function
    x(2) # 5
```
